夜にインシデントをチェックしている、または目に負担をかけたくない場合、UIをダークモードで表示できます。

![APM_distributed_tracing_dark-Mode.png](./images/APM_distributed_tracing_dark-Mode.png "APM_distributed_tracing_dark-Mode.png")

<figcaption>
  New RelicのUIでライトモードとダークモードを切り替えるには、アカウントドロップダウンをクリックします。
</figcaption>

## ライトモードとダークモードを切り替える [#dark-mode]

ダークモードでUIを表示するには：

1. [one.newrelic.com](https://one.newrelic.com)に移動し、[アカウントドロップダウン](/docs/using-new-relic/welcome-new-relic/get-started/glossary#account-dropdown)をクリックします。\[注: 現在、ダークモードはFirefoxブラウザではサポートされていません。]

   ![icon-plus.png](./images/account_dropdown_theme.png "account_dropdown_theme.png")
2. **テーマ:**をクリックし次のオプション間で切り替える：

   * **テーマ:自動**（デフォルト: UIを、OSですでに有効にしているものと一致させます。
   * **テーマ:ダーク**: UIをダークモードにします。
   * **テーマ:ライト**: UIをライトモードにします。
